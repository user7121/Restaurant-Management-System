# Smart Restaurant Management System

## 📌 Project Overview

This repository contains the **Smart Restaurant Management System** project prepared for the **Software Project Management** course.
The project focuses on applying Agile project management practices, including Jira-based sprint planning, documentation, UML modeling, and structured project deliverables.

## 🎯 Project Objectives

* Develop a conceptual restaurant management system
* Apply Software Project Management lifecycle
* Use Agile methodology (Scrum, Sprints, Backlog)
* Prepare academic project documents (Excel, UML, Budget, Timeline)

## 🛠 Project Management Approach

The project is managed using:

* Jira (Epics, Stories, and Sprint Planning)
* Agile Scrum methodology
* 14-week timeline aligned with course syllabus
* Task distribution among Backend, Frontend, and Database roles

## 📊 Repository Contents

* `docs/` → Project documentation files (Excel, UML, planning documents)
* `README.md` → Project description and overview
* Jira Planning → Sprint structure, backlog, and task management (shared with instructor)

## 📅 Project Planning Deliverables

This repository includes all required course deliverables:

* Excel Planning File (Team members, roles, timeline, and budget)
* UML Diagram (System Use Case)
* GitHub Repository Organization
* Jira Task Management & Sprint Planning

## 📈 Agile & Sprint Structure

The project is divided into multiple planned sprints:

* Sprint 1: Authentication & Core Setup
* Sprint 2: Menu & Product Management
* Sprint 3: Order & Table Management
* Sprint 4: Reporting Features
* Sprint 5: Financial Analysis
* Sprint 6: Documentation & Final Deliverables

(Note: Sprints are planned but not prematurely started to align with academic project planning requirements.)
